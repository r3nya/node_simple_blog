# Пример блога

Студент, помни!

В начале установи `bower` глобально:
```shell
npm install -g bower
```

## Деплой проекта

```shell
git clone https://github.com/r3nya/node_simple_blog.git
npm install
bower install
```

## Запуск

```shell
npm start
```
